### 주요 회의 내용
1. **GAIA 기능 3가지 재설계**  
   - 기능 ① 테스트 설계 자동화  
   - 기능 ② 적용형 자동 실행  
   - 기능 ③ 테스트 실패 세부 정보 제공  
   - 각 기능별 목표와 세부 기능 요구사항을 정리하고 발표용 문서에 반영하기로 함  

2. **웹 테스트 요소 조사**  
   - 버튼, 입력창, 체크박스, 팝업 등 **웹에서 테스트 가능한 모든 요소**를 조사 완료  
   - 조사 결과를 기반으로 향후 자동화 대상 확장을 검토
